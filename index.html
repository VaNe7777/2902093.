What is Java?

The first is the first. We have to talk about what Java is, because, whether you already have control or are just starting out,
it is always a good idea to contextualize what we are going to talk to you about.

Java is a programming system that serves both to develop applications and to support those applications. That is, it is a full stack programming language,
which serves to shape what users see and also to support the server part, which makes everything really work.
